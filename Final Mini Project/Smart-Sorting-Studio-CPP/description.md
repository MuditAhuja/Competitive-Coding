# Smart Sorting Studio

Smart Sorting Studio is a comprehensive educational and benchmarking tool designed to analyze and compare various sorting algorithms. The project evaluates sorting techniques based on multiple performance metrics, including time complexity, swap counts, comparison operations, and algorithm stability, running against different dataset scenarios (e.g., student marks, sales data, and system logs).

It features two primary components:
1. **A Core C++ Engine**: A highly efficient terminal-based application that executes the algorithms (such as Merge Sort, Quick Sort, Heap Sort, Counting Sort, Radix Sort, and Bucket Sort) and computes precise performance metrics.
2. **A Full-Stack Web Interface**: A modern web application featuring a React (Vite) frontend and a Node.js backend API, allowing users to interactively test algorithms and visualize the sorting processes in an intuitive environment.
